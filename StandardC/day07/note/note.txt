指针变量是用来记录地址数据的，在我们的计算机
里所有指针变量占据4个字节的空间。声明指针
变量的时候要使用符号*来描述。

int shu_zi;  //整数变量的声明
int *p_shu_zi;  //指针变量的声明
char *p_zi_fu;  //指针变量声明的时候要把它所记录
                //的地址来源的变量的类型描述出来

指针变量必须使用NULL进行初始化，NULL对应的地址
数据是0，我们把它叫做空地址。记录空地址的指针
变量叫做空指针。

局部变量的地址不能作为返回值使用。

在编程对数组进行处理的时候完全可以把指针变量
当成数组名称使用。指针变量还可以使用自增的
方式完成对数组中每个变量地址的列举过程。

在c语言程序中可以使用一组连续的字符表示文字
信息，我们把这些字符构成的整体叫做字符串。
字符串在计算机里只能使用首字符的地址来表示。
在程序中可以使用双引号包含起来的一组字符表示
字符串，这种表示方式叫做字符串字面值。程序
中还可以用一个字符数组来表示一个字符串。这
两种方式表示的字符串在运行中采用不同的方式
进行管理。

main函数也有参数，参数格式如下
int main(int argc, char* argv[]) {
}
其中argv是一个数组，数组中每个变量是一个字符串
指针。用户在执行命令的时候输入的所有内容被依据
空格字符进行了分组，每组对应一个字符串。所有这些
字符串的地址被按顺序放在参数argv数组中。参数
argc表示参数argv中有多少个可用的字符串。
